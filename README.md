## Installation
Via curl
```bash
bash -c
```

Manually
1. Copy file gusrylmubarok.zsh-theme to $ZSH_CUSTOM/themes folder. (You may need to create `themes` folder if it does not exist)
2. Set theme to gusrylmubarok in ~/.zshrc :
```bash
ZSH_THEME="gusrylmubarok"
```
